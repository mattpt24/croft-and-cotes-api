Welcome to the official Croft & Cotes API

Created by Matt Persell-Thompson

Hi, this is a free open source API i've created to practice building and interacting with API's. It is an API of songs from mine and my twins band called Croft & Cotes. Happy coding!

All Songs.

Return all 15 songs
Useage
https://songs-api-puce.vercel.app/songs

Random Song.

Return a random song on refresh.
Useage
https://songs-api-puce.vercel.app/random

Specific Song.

Get a specfic song by ID. There are 15 songs in total
Useage
https://songs-api-puce.vercel.app/song/15

Filter Songs.

You can filter songs by year and/or author
Useage
https://songs-api-puce.vercel.app/filter?year=2020&author=Matt
